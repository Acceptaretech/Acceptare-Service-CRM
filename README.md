# Lead Management Backend

## Description
This is a backend project built using Node.js, Express, and MongoDB for managing leads.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features
- Create a lead (POST)
- Get all leads (GET)
- Update lead status (PUT)
- Delete a lead (DELETE)

## API Endpoints
- POST `/api/leads`
- GET `/api/leads`
- PUT `/api/leads/:id`
- DELETE `/api/leads/:id`
